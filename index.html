What is the crawl ratio?
Crawl ratio is a term used in automotive world to describe the highest gear ratio that a vehicle is capable of.
How to calculate crawl ratio?
To calculate crawl ratio you should use the following formula:
crawl ratio = TR * TCR * AGR
where:
TR = transmission first gear ratio
TCR = transfer case low gear ratio
AGR = axle gear ratio
Why use crawl ratio?
The crawl ratio is used in order to describe the highest gear ratio that a vehicle is capable of.
